# Outfit Recommendation System V3 

## Overview
Version 3 improves the recommendation engine by introducing feature scaling and normalized similarity calculations.

The system now compares outfit features more fairly by scaling:
- Dominant colors
- Brightness
- Contrast

This produces more balanced clustering and more accurate outfit recommendations.

## New Features in V3

- More balanced recommendations
- Cleaner recommendation pipeline
