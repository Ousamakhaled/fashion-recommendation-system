# Outfit Recommendation System V2 

## Overview
This version improves the original outfit recommendation system by introducing additional visual features beyond dominant colors.

The system now analyzes:
- Brightness
- Contrast
- Dominant color patterns

This leads to more accurate fashion similarity recommendations.

---

## Technologies Used
- Python
- NumPy
- Matplotlib
- Scikit-learn
- Pillow (PIL)

---

## New Features in V2
- Brightness extraction
- Contrast analysis
- Improved outfit similarity matching
- Named fashion style groups
- Better recommendation quality

---

## Style Groups

### Neutral / Minimal
Clean outfits with soft neutral tones.

### Dark / Streetwear
High-contrast darker outfits with streetwear aesthetics.

### Light / Casual
Bright and relaxed casual fashion styles.
