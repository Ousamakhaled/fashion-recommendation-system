# Outfit Recommendation System V4 

## Overview
This version improves the recommendation system by adding:

- Edge detection
- Texture analysis
- Brightness features
- Contrast features

The goal was to improve outfit similarity matching beyond only dominant colors.

---

## Technologies Used
- Python
- NumPy
- OpenCV
- Scikit-learn
- Matplotlib
- Pillow (PIL)

---

## New Features in V4
- Edge density extraction
- Texture feature extraction
- Improved similarity scoring
- Better style clustering
- More accurate outfit recommendations
